Visual Builder for Cordova

(Hybrid mobile application framework)

Introduction

  • Apache Cordova is a platform for building natively installed mobile applications using HTML,CSS and JavaScript
  • There are many other frameworks and tools built on top of Cordova including Ionic, Monaca, TACO, the Intel XDK and the Telerik Platform.

Cont'

  • These tools and frameworks provides their own components to design the applications and these components are mostly provided as code segments or sometimes as a part of visual editor

Cont'

  • There is no visual builder that works with just Cordova and this has been a problem for those who want to build applications with pure Cordova.

Objectives

  • Speeding up the development procedure by providing a visual builder
  • Make the tool popular among non-developers, specially designers
  • Making a visual builder for Cordova which is free and open source
  • Reducing the limitations for the designers, despite of their programming knowledge
